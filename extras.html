Van een simpel Virtual Reality klimbos naar een volledige mobile game tot een app om je Morse Code te leren. Bijna alle projecten die ik heb gemaakt om mijn skills te verbeteren en waar in ik ook iets heb geleerd staan hier.

Projecten:

Gem Ride

In deze mobile game moet je proberen zo lang mogelijk te overleven over de rails, terwijl je steeds sneller en sneller zal gaan. Ook kan je power-ups kopen en skins voor je player.

Crystal Crash

Voor een school gamejam heb ik samen met 4 andere een korte race game gemaakt. Het thema was Mirror, dus tijdens je race reis je naar een spiegel dimensie. Ook zijn er andere auto's die tegen je strijden.
