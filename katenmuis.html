Kat & Muis (Stage project is nog gaande)

Kat & Muis is een spannend spel waarin de kat (Speler in VR) de muizen (Spelers op mobiel) moet proberen te vangen, dit allemaal terwijl de muizen kaas proberen te verzamelen en veilig stellen. Kan jij je vrienden vangen?

Korte Info:

  • Gemaakt met: Unity
  • Codeertaal: C#
  • Mijn rollen: Developer, Sound Designer, Scrum Master
  • Tijdspan: 8 weken

Team:

    Developers:

  • Didier Vermunt

Mijn Bijdragen:

Network Room System:

Via Mirror en Noble Connect heb ik een multiplayer room systeem gemaakt. Hierdoor kan de speler in vr een room aanmaken, en een speler op mobiel kan deze lobby kiezen uit een lijst en dan joinen. Voor de lobby gebruik ik een MySql database die beheert wordt door mijn stage bedrijf.

Nav Mesh VR Teleport:

Na meerdere playtests bleek dat mensen snel misselijk werden door de movement, hiervoor heb ik de keuze gemaakt om teleportation te gebruiken i.p.v. continuous movement. Alleen door de abstracte layout van de map kan ik niet zo maar één plane plaatsen, hiervoor heb ik een script gemaakt dat het teleporten van VR alleen maar mogelijk maakt op een navmesh surface.

Respawn System:

Een muis kan 3 keer gevangen worden door de kat voordat deze af is, je wilt alleen natuurlijk niet weer op dezelfde plek terug komen, misschien dat de kat je dan weer in één te vangen heeft. Daarom heb ik een script gemaakt dat de speler terugplaatst in een andere 'sector' in de map.
